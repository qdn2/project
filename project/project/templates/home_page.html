About The Project

This project is designed to enable a user to search for a recipe or restaurant. A user can navigate to either search page depending on what they want to find. The user can search a restaurant based off of the exact restaurant name, city name, or by a substring of the restaurant's name. Furthermore, once the user has naviagated to a particular restaurant page there is a recommendation functionality at the bottom of the page which leads the user to a page of recipes that are related to the type of food that the particular restaurant serves. On the recipe end of things, the user can search for recipes based of off exact name of the recipe, partial name of the recipe, ingredients, or by the cuisine type.
